## Обзор игры

**CYBERWAR** — это динамичный аркадный шутер с видом сверху, действие которого разворачивается на закрытой арене. Игроку предстоит противостоять волнам враждебных роботов, используя тактическое перемещение и точную стрельбу. Проект включает как одиночный режим выживания, так и основы мультиплеера по локальной сети.

## Ключевые особенности

- **Два типа противников:**
  - *Ближний бой (Melee):* Быстрые, агрессивные юниты, стремящиеся сократить дистанцию.
  - *Дальний бой (Ranged):* Держат дистанцию, атакуют снарядами, уклоняются от игрока.
- **Система комбо-множителя:** Чем быстрее игрок уничтожает врагов, тем выше множитель очков. Сброс множителя происходит при получении урона или слишком долгом перерыве между убийствами.
- **Базовый мультиплеер (Mirror):** Синхронизация движения игроков по локальной сети (LAN).
- **Полный цикл игровых состояний:** Игрок имеет здоровье, может лечиться, ставить игру на паузу, умирать и перезапускать сессию.

## Геймплейные механики

| Механика | Описание |
| :--- | :--- |
| **Здоровье (HP)** | Полоска здоровья, отображаемая на экране. При достижении нуля игрок погибает. |
| **Лечение** | На карте генерируются аптечки, восстанавливающие фиксированное количество HP. |
| **Система очков** | Очки начисляются за каждого уничтоженного врага. Базовая стоимость убийства умножается на текущий множитель. |
| **Комбо-множитель** | Увеличивается на 0.5 за каждое убийство. Множитель отображается в UI и постепенно уменьшается, если игрок не убивает врагов. |
| **Пауза** | Возможность поставить игру на паузу с выводом соответствующего меню. |
| **Смерть и возрождение** | При смерти игрока останавливается подсчет очков и появляется экран с результатами и предложением начать заново. |

## Технические детали

- **Движок:** Unity 2022.3.47f1
- **Язык программирования:** C#
- **Архитектура:** Компонентно-ориентированный подход (ECS-like через MonoBehaviour)
- **Сеть:** Mirror Networking (транспортный уровень KCP / Telepathy)
- **Управление:** Ввод с клавиатуры (WASD) и мыши (прицел / стрельба)

## AI противников

Искусственный интеллект реализован с использованием встроенной системы **NavMesh** Unity, что обеспечивает естественное перемещение врагов по уровню.

- **Механика поведения:**
  1. **Поиск пути (Pathfinding):** Враги используют `NavMeshAgent` для навигации по статической геометрии уровня.
  2. **Состояния (State Machine):** Каждый враг имеет простую машину состояний: `Idle` (патруль), `Chase` (преследование), `Attack` (атака).
  3. **Различия в поведении:**
     - *Melee:* Постоянно движется к игроку, атакует в упор.
     - *Ranged:* Держит дистанцию (если игрок подходит слишком близко — отступает), останавливается на оптимальной дистанции для стрельбы.
- **Спавн:** Враги спавнятся волнами из заранее определенных точек на карте.

## Сетевая реализация (Mirror)

В проекте реализована базовая клиент-серверная модель с использованием библиотеки Mirror.

- **Топология:** Клиент-сервер (один из игроков выступает в роли хоста).
- **Синхронизация:**
  - Позиции и повороты игроков синхронизируются по сети (используются NetworkTransform).
  - Спавн игроков при подключении.
- **Компоненты:**
  - `NetworkManager` — управление подключениями и сценами.
  - `NetworkIdentity` / `NetworkTransform` — сетевые объекты игроков.
  - `NetworkBehaviour` — пользовательские скрипты, взаимодействующие с сетью.
- **Текущее состояние:** Синхронизация движений работает стабильно. Механики стрельбы, здоровья и взаимодействия с врагами в сетевом режиме находятся в стадии доработки.

## Установка и запуск

### Для игроков
1. Скачать последний релиз из раздела [Releases](https://github.com/username/CYBERWAR/releases).
2. Распаковать архив.
3. Запустить `CYBERWAR.exe`.

### Для разработчиков (самостоятельная сборка)
1. Клонировать репозиторий:
   ```bash
   git clone https://github.com/username/CYBERWAR.git
   ```
2. Открыть проект в Unity Hub (версия 2021.3 или новее).
3. Убедиться, что пакет Mirror установлен (Window -> Package Manager -> My Assets -> Mirror).
4. Открыть сцену `Assets/Scenes/Main.unity`.
5. Нажать `Play` для тестирования в редакторе.

### Для мультиплеера
1. Собрать билд игры (File -> Build Settings).
2. Запустить первый экземпляр игры и нажать "Host".
3. Запустить второй экземпляр (на другом компьютере в локальной сети) и ввести IP-адрес хоста в поле "Address", нажать "Client".
